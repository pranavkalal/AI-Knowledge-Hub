# Benchmark Query Set

## Overview

This document defines the baseline set of queries used for A/B testing and evaluation of different configurations. All experiments must use this query set to ensure consistent, repeatable, and comparable results. The queries are designed to evaluate performance across multiple reasoning types, including:
- factual retrieval
- relational reasoning
- multi-hop reasoning
- thematic analysis
- open-ended queries

---

# Query Set

## Factual Queries

1. What yield improvement was observed in cultivar Sicot 746B3F?
2. Which study investigated bollworm resistance in cotton?
3. What year was the CRDC irrigation study conducted?
4. What methods are used to measure cotton yield in field trials?
5. Which cotton varieties were tested for drought resistance?

## Relational Queries

6. How does irrigation impact cotton yield?
7. What factors influence drought resistance in cotton?
8. How are pest management strategies linked to yield outcomes?
9. What is the relationship between soil quality and cotton productivity?
10. How do fertiliser practices affect crop performance?

## Multi-Hop Queries

11. Which cotton varieties have shown improved drought resistance in field trials?
12. What research connects irrigation practices to yield improvement?
13. Which studies link soil quality to pest resistance?
14. How do different environmental factors interact to affect cotton yield?
15. What evidence exists linking water usage to long-term crop performance?

## Thematic/Global Queries

## Ambiguous/Open-Ended Queries

# Notes

This query set is designed to:
- reflect realistic user queries
- test a range of reasoning capabilities
- highlight differences between retrieval strategies

The set may be expanded in future iterations, but any changes must be version-controlled.
